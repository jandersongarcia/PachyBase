# PachyBase Roadmap

This roadmap is directional and may evolve as the product grows.

## Stable today

- Predictable JSON API contract
- Docker-first local setup
- MySQL and PostgreSQL support
- Migrations and seeds
- JWT and API token authentication
- Automatic CRUD
- OpenAPI generation
- AI-friendly discovery endpoints
- Project CLI
- Automated regression suite

## Next

- Better API exploration on top of the generated OpenAPI document
- Richer CRUD filtering
- More integration examples
- Stronger deployment and maintenance guides

## Later

- More extension points around hooks and events
- Advanced authorization and audit capabilities
- Optional SDK generation workflows
- More operational tooling

See the documentation version at <https://jandersongarcia.github.io/pachybase/roadmap>.
