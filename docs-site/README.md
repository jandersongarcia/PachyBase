# PachyBase Docs

This site is built with [Docusaurus](https://docusaurus.io/) and documents PachyBase in two locales:

- `en`
- `pt-BR`

## Installation

```bash
npm install
```

## Local development

```bash
npm run start
```

## Build

```bash
npm run build
```

## Translation structure

- Default English docs live in `docs/`
- Brazilian Portuguese docs live in `i18n/pt-BR/docusaurus-plugin-content-docs/current/`
