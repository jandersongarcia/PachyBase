# Roadmap

Este roadmap e direcional, nao uma promessa fechada de release. Ele mostra onde o PachyBase ja esta solido e para onde o projeto deve crescer em seguida.

## Estavel hoje

- contrato JSON previsivel para a API
- fluxo de instalacao Docker-first
- adapters para MySQL e PostgreSQL
- migrations e seeds
- autenticacao com JWT e API tokens
- CRUD automatico com configuracao declarativa por entidade
- documento OpenAPI gerado
- endpoints de descoberta para IA
- CLI do projeto e suite de regressao

## Proximo

- experiencia de exploracao da API em cima do OpenAPI gerado
- operadores de filtro mais ricos alem de igualdade simples
- mais exemplos cobrindo modulos reais e integracoes
- guias melhores de deploy e manutencao

## Depois

- pontos de extensao mais amplos para eventos e hooks
- autorizacao mais avancada e capacidades de auditoria
- fluxos opcionais de geracao de SDK
- mais ferramental operacional para backup, restore e diagnostico
