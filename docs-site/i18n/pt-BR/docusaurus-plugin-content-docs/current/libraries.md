# Bibliotecas

Esta pagina lista as principais bibliotecas atualmente usadas pelo PachyBase e pelo site de documentacao.

## Runtime

- [vlucas/phpdotenv](https://github.com/vlucas/phpdotenv): carrega variaveis de ambiente a partir do `.env`.

## Site de documentacao

- [Docusaurus](https://docusaurus.io/): framework de documentacao usado pelo `docs-site`.
- [React](https://react.dev/): biblioteca de interface usada pelo Docusaurus.
- [TypeScript](https://www.typescriptlang.org/): tipagem estatica para a configuracao da docs.
- [prism-react-renderer](https://github.com/FormidableLabs/prism-react-renderer): destaque de sintaxe para blocos de codigo.
- [MDX](https://mdxjs.com/): suporte a Markdown com JSX nas paginas de documentacao.

## Referencias no codigo-fonte

- Dependencias do backend: [`composer.json`](../../../composer.json)
- Dependencias da docs: [`package.json`](../package.json)
