# Roadmap

This roadmap is directional, not a strict release promise. It shows where PachyBase is already solid and where the project is expected to grow next.

## Stable today

- predictable JSON API contract
- Docker-first installation flow
- MySQL and PostgreSQL adapters
- migrations and seeds
- JWT and API token authentication
- automatic CRUD with declarative entity configuration
- generated OpenAPI document
- AI-friendly discovery endpoints
- project CLI and regression test suite

## Next

- bundled API explorer experience for the generated OpenAPI document
- richer filter operators beyond simple equality
- more examples covering real feature modules and integration patterns
- improved deployment and maintenance guides

## Later

- broader extension points around events and hooks
- more advanced authorization and audit capabilities
- optional SDK generation workflows
- additional operational tooling around backup, restore, and diagnostics
