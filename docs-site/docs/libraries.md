# Libraries

This page lists the main libraries currently used by PachyBase and its documentation site.

## Runtime

- [vlucas/phpdotenv](https://github.com/vlucas/phpdotenv): loads environment variables from `.env`.

## Documentation site

- [Docusaurus](https://docusaurus.io/): documentation framework used by the `docs-site`.
- [React](https://react.dev/): UI library used by Docusaurus.
- [TypeScript](https://www.typescriptlang.org/): static typing for the docs configuration.
- [prism-react-renderer](https://github.com/FormidableLabs/prism-react-renderer): syntax highlighting for code blocks.
- [MDX](https://mdxjs.com/): Markdown with JSX support for documentation pages.
